Using VITE
Create Project (using vite instead of CRA like in tutorial, it is outdated)
  bun create vite@latest typescript-tutorial --template react-ts
  cd typescript-tutorial
  bun install

Run Project (NOTE: the starting commands AND the ports are different than you are used to)
  bun run dev  (starts the dev)
    http://localhost:5173/
  bun run build  (builds the app)
  bun run preview  (runs the built app)
    http://localhost:4173/
